# RxDB node example

This is an example usage of RxDB with node. It implements a simple heroes-list.

# Try it out
1. clone the whole [RxDB-repo](https://github.com/pubkey/rxdb)
2. go into project `cd rxdb`
3. run `npm install`
4. go to this folder `cd examples/node`
5. run `npm install`
6. run `npm start`
